Detailed Description

Program that tracks the face of the subject that appears on the camera for the registration process (ticketline simulation). 
It will also calulate the descriptors that define the face presented and, if needed, it will send such descriptors to the verification server.
Finally, the program will send a signal to the camera acquisition program if the person changes for a further calibration.

It will divide the people that appear into the camera into several videos.

Once a person stops appearing on the camera and another one starts to appear, the function will stop recording in the current file and start to record on a different one. It will also stop recording if the camera does not detect any face in a given period of time.

frameFrame acquired by the camera
dThe comparision value between descriptors (to know if it is not the same person)
